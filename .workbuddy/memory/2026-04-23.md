## 修复：充电桩列表连接状态 Bug
- 问题：点击任意设备"连接"时，列表中所有设备的按钮都显示 loading 转圈
- 原因：`_DeviceCard` 仅用全局 `isConnecting.value` 判断，未区分是否为当前卡片
- 修复：
  1. `ChargerController` 新增 `connectingDeviceId = Rxn<String>()` 字段，连接开始时记录目标设备 remoteId，结束后清空
  2. `_DeviceCard` 改用 `isThisDeviceConnecting = ctrl.isConnecting.value && ctrl.connectingDeviceId.value == result.device.remoteId.str` 判断，只有目标设备卡片显示 loading
