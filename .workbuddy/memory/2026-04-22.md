# 2026-04-22 工作日志

## 充电桩功能修复

### 修复 charger 首页主题适配问题
- **问题**：charger_page.dart 中大量使用硬编码颜色（如 `Color(0xFFF5F6FA)`, `Colors.grey[300]` 等），没有根据亮色/暗黑模式自适应
- **修复方案**：
  1. 统一使用 `Theme.of(context).colorScheme` 获取主题颜色
  2. 替换硬编码颜色为语义化颜色（`surface`, `primary`, `error`, `outline` 等）
  3. Scaffold 背景色改为 `colorScheme.surface`
  4. 卡片背景改为 `colorScheme.surfaceContainerHighest`
  5. 状态卡片渐变色适配：`connected` 时使用 `primary/secondary`，`disconnected` 时使用 `outline/outlineVariant`

### 添加模拟测试开关
- **需求**：用户希望能通过 UI 切换模拟模式，无需修改代码即可切换真实/模拟业务
- **实现**：
  1. 将 `ChargerController.isMock` 从 `final bool = true` 改为 `RxBool(true.obs)`
  2. 在 AppBar 添加 `Switch` 开关，实时切换 `isMock` 状态
  3. 更新所有引用 `isMock` 的地方为 `.value` 访问
  4. 扫描按钮、Mock 区域、设备列表根据 `isMock.value` 动态显示/隐藏
  5. Mock 模式提示信息更新，告知用户通过 AppBar 开关切换模式

### 修改文件
- `lib/pages/charger/charger_controller.dart` - isMock 改为可观察变量
- `lib/pages/charger/charger_page.dart` - 主题适配 + 模拟开关 UI

## 首页 Tab 懒加载优化

### 需求
- 打开首页时只加载当前 Tab 内容，其他 Tab 不预加载
- 点击到某 Tab 时才开始加载内容
- 第二次点击同一 Tab 不重复加载（内容缓存）

### 实现方案
1. **HomePage 改造**
   - 用 `Stack` + `Offstage` 替代 `IndexedStack`
   - 用 Map 存储各 Tab 的加载状态 `_tabLoaded`
   - 首次点击 Tab 时标记为已加载
   - 未加载的 Tab 在底部导航显示小红点提示

2. **ShortVideoTab 视频懒加载**
   - 添加 `VisibilityDetector` 检测视频 Feed 是否可见
   - `_VideoPage` 支持 `lazyLoad` 参数
   - 不可见时不初始化视频播放器，节省性能

### 修改文件
- `lib/pages/home/home_page.dart` - Tab 懒加载架构
- `lib/pages/home/tabs/short_video_tab.dart` - 视频懒加载 + VisibilityDetector
